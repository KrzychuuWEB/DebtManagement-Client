## Description
This project helps for debt managment. 
I used to create this project:
- Java / Spring boot
- Javascript / React
- MySQL
- JUnit
- Mockito
- Integration test
- Rest API
- Hibernate
- Liquibase

## Features:
- View all debtors
- View all unpaid debts
- CRUD debtor
- CRUD debts
- Debt repayment
